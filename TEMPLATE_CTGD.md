# Hướng dẫn tạo file Excel CTGD (Chương trình giảng dạy)

File Excel CTGD cần có định dạng như sau:

## Cấu trúc cột:

| Môn học | Tiết thứ | Tên bài |
|---------|----------|---------|
| Toán | 1 | Ôn tập các số đến 100 |
| Toán | 2 | Số hạng - Tổng |
| Toán | 3 | Đề-xi-mét |
| Tiếng Việt | 1 | Tập đọc: Có công mài sắt, có ngày nên kim |
| Tiếng Việt | 2 | Chính tả: Nghe - viết: Có công mài sắt, có ngày nên kim |
| Tiếng Việt | 3 | Tập đọc: Tự thuật |

## Lưu ý:

- Cột đầu tiên phải là "Môn học" - tên môn học (ví dụ: Toán, Tiếng Việt, Đạo đức...)
- Cột thứ hai là "Tiết thứ" - số thứ tự tiết học trong năm học (1, 2, 3, ...)
- Cột thứ ba là "Tên bài" - tên bài dạy tương ứng
- Hệ thống sẽ tự động khớp: Môn học + Tiết thứ trong năm = Tên bài
- Lưu file dưới định dạng .xlsx hoặc .xls
